Windows SDK exe installation¶

Windows 10

Download and install opencv3.4.3¶

opencv-3.4.3-vc14_vc15.exe

Setting environment variables¶

  • Setting OpenCV path: OPENCV_DIR = <opencv>\build
  • Add OpenCV directory to PATH: %OPENCV_DIR%\x64\vc15\bin

Tip

<opencv> indicates the install path

Download and install SDK¶

mynteye-2.0-win-x64-opencv-3.4.3.exe

Run samples¶

$ c:
$ cd Program Files (x86)\Slightech\MYNT EYE SDK 2.0
$ .\samples\bin\api\camera_a.exe
